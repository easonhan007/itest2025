[官方文档](http://docs.python-requests.org/en/master/user/install/#install)

官方给出的安装方式是使用[pipenv](http://pipenv.org/)

```
pipenv install requests
```

当然也可以直接使用pip安装

```
pip install requests
```

还可以从源码安装

```
git clone git://github.com/requests/requests.git
cd requests
pip install .
```
